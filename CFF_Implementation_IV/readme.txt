\f0\fs24 \cf0 This folder contains the companion code for the paper by:\
\
Centorrino, Feve, Florens: Additive Nonparametric Instrumental Regressions: A guide to the implementation, Apr 2015\
\
List of files:\
\
1) \'93simulations_final.m\'94 runs the Monte-Carlo simulations of section 4.\
2) \'93Engel_curve_pakistan_data.m\'94 contains the code to run the empirical example of section 5. Notice that some of the estimators require a running time of more than an hour, unless users provide rule of thumb bandwidths, instead of the CV bandwidths used so far.\
3) \'93ba_data.csv\'94 contains the subsets of the data in Bhalotra and Attfield (1998) used in the empirical application.\
4) The folder \'93aux\'94 contains the auxiliary files used for estimation; selection of the CV bandwidths and CV regularization parameter. \
\

\b Please notice that the code is in beta, until further notice. \
